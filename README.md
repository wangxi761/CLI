# CLI

## install

```
dotnet tool install --global Tars.Net.CLI --version 0.0.1-beta-20181021132156
```

## Commands

``` shell
Tars.Net Command Line Tool

Usage: dotnet tarsnet [options] [command]

Options:
  -?|-h|--help  Show help information
  --version     Show version information

Commands:
  codecs  Generate tars file to csharp file

    -- Usage: dotnet tarsnet codecs [arguments] [options]
    -- Arguments:
        Source                 Tars file or Tars file directory
        Destination directory  CSharp file destination Directory


```
